vite is cool

Before ES modules were available in
browsers, developers had no native mechanism
for authoring JavaScript in a modularized fashion. This is why we
are all familiar with the concept of "bundling":
using tools that crawl, process and
concatenate our source modules into
files that can run in the browser.
Over time we
have seen tools like webpack,
Rollup and Parcel, which greatly
improved the development experience
for frontend developers.
